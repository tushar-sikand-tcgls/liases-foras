🎯 8 STATISTICAL OPERATIONS INCLUDED:
═════════════════════════════════════════════════════════════════════════════════════════

1. TOTAL (SUM)
   └─ Aggregate all values
   └─ Use: Total market supply, revenue
   └─ Formula: Σ(x_i)

2. AVERAGE (MEAN)
   └─ Arithmetic mean value
   └─ Use: Typical project size, price
   └─ Formula: Σ(x) / n

3. MEDIAN
   └─ Middle value (50th percentile)
   └─ Use: Typical without outliers
   └─ Formula: Sorted middle value

4. MODE
   └─ Most frequent value
   └─ Use: Common price range, unit type
   └─ Formula: Highest frequency

5. STANDARD DEVIATION
   └─ Measure of spread/volatility
   └─ Use: Risk assessment, consistency
   └─ Formula: √[Σ(x-μ)²/(n-1)]

6. VARIANCE
   └─ Squared deviation (before sqrt)
   └─ Use: Statistical basis, regression
   └─ Formula: Σ(x-μ)²/(n-1)

7. PERCENTILE
   └─ P% of data below this value
   └─ Use: Top performers, benchmarking
   └─ Formula: Sorted position calculation

8. NORMAL DISTRIBUTION (BELL CURVE)
   └─ Gaussian curve fit & outlier detection
   └─ Use: Probability analysis, outlier flagging
   └─ Formula: Gaussian PDF + Z-scores
